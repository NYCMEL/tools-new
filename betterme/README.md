# betterme

Production-ready BetterMe-style onboarding quiz built with HTML, SCSS, and Vanilla JavaScript ES6.

## Files

- `index.html`
- `partials/betterme.html`
- `js/_wc.js`
- `js/betterme.config.js`
- `js/betterme.js`
- `scss/betterme.scss`
- `css/betterme.css`
- `tmp/` sample images

## Run

Open with a local server so `wc-include` can fetch the partial.

```bash
python3 -m http.server 3000
```

Then open:

```text
http://localhost:3000
```

## Compile SCSS

```bash
sass scss/betterme.scss css/betterme.css
```
